Quick Help

Create your first Web Shortcut
  • Open or reload a website.
  • Create a Web Shortcut: Use Ctrl/Command + Shift + Space or the popup's plus (+) button.
  • Drag the Element Selector anywhere on the page.
  • Read the 3 ways to select an Element inside the Element Selector.
  • Hover over elements to see a red border; select one using the Element Selector.
  • Choose a single occurrence Element (avoid multiples like social media 'like' buttons).
  • Key Selector opens; press a character key for your shortcut.
  • Give the shortcut a fitting name.
  • Refer to the usage guide for additional properties if needed.
  • For Input Elements, select "Focus" as the Action.
  • Click "Add Shortcut" or press Enter.
  • Keep the Element Selector open for more selections.
  • Close the Element Selector when finished.
  • Test your shortcut on the website.

Quick Tips
You don't need the mouse for most of the actions!
  • Use Ctrl/Command + Shift + Space to quickly create a new shortcut.
  • Use the Esc key to close Key Selector or Element Selector.
  • Use the Enter key to Add shortcut without clicking on the "Add Shortcut (+)" button in the Key Selector.
The Web Shortcuts I created aren't working
  • Ensure your chosen shortcut isn't already in use on the website by pressing the desired key once there, before creating the shortcut.
  • Confirm you've selected the correct element.
  • Try reloading the tab, or closing and opening the Tab again.
  • Disable certain extensions that might cause issues,, reload the tab, and try again.
  • Ensure the urlType & action properties are correctly set according to your needs.
  • Attempt selecting the element again with a new shortcut. Delete the previous one and edit the new in the options page.
Unable to select an Element
  • Reload the Tab
  • Disable other extensions that may be causing issue
  • The website itself can be causing this problem. Share this with us on Discord and we will look into it.
Element can't be found, please try selecting again
  • This happens when the selected element is removed from the DOM during the process of creating the web shortcut.
  • Try selecting the element using other ways as mentioned in the Element Selector.
  • Hovering on the Element and Pressing Enter usually works.
Element was not found
This message is shown when the Element is not present in the DOM.
  • Try checking if the element is present inside some other element.
  • If that is the case, try creating a shortcut to open it.
  • This will let you open it and access the desired element.
  • This might seem a bit annoying, but we can't work around it as we can't access the Element if it doesn't exist.
The extension is not responding
Try reloading the tab and the extension once.
Extension shortcuts aren't working
  • Try Disabling other extensions.
  • On some websites, Esc or other shortcut keys that are used to operate the extension might not work.
