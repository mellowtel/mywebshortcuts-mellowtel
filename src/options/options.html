v1.0.0

My Web Shortcuts is an open-source browser extension that lets you create custom keyboard shortcuts in any website to click (or perform other actions) on any element(s).

Note: This extension is currently in it's early development phase, you may face a few bugs and lack of features. If you face an issue, kindy report it to us immediately.
